# Plan: Move Arcjet WebSocket Protection to HTTP Upgrade Handler

## Objective
Move Arcjet protect() call from inside `wss.on('connection', ...)` to the HTTP server's 'upgrade' event to validate WebSocket handshake before it's established.

## Changes Required

### 1. `src/ws/server.js` - Modify attachWebSocketServer function

**Current flow:**
- `wss.on('connection', ...)` handler performs Arcjet protection AFTER connection established

**New flow:**
- Remove Arcjet code from `wss.on('connection', ...)` (lines 26-43)
- Add `server.on('upgrade', ...)` handler to intercept WebSocket upgrade requests
- Inside upgrade handler:
  - Call `wsArcjet.protect(req)` (if wsArcjet exists)
  - If denied: write HTTP error response (403) and destroy the raw socket
    - Rate limit → `429 Too Many Requests`
    - Access denied → `403 Forbidden`
  - If allowed: call `wss.handleUpgrade(req, socket, head, ws => wss.emit('connection', ws, req))`
  - Catch block: destroy raw socket on error

### 2. Code Changes Detail

**Remove from wss.on('connection'):**
```javascript
// REMOVE lines 26-43:
if (wsArcjet) {
  try {
    const decision = await wsArcjet.protect(req);
    if (decision.isDenied()) {
      const code = decision.reason.isRateLimit() ? 1013 : 1008;
      const reason = decision.reason.isRateLimit()
        ? "Rate Limit Exceeded"
        : "Access Denied";
      socket.close(code, reason);
      return;
    }
  } catch (e) {
    console.error("WS connection error: ", e);
    socket.close(1011, "Server Security Violation");
    return;
  }
}
```

**Add to attachWebSocketServer (after wss creation):**
```javascript
server.on("upgrade", async (req, socket, head) => {
  if (!wsArcjet) {
    wss.handleUpgrade(req, socket, head, (ws) => wss.emit("connection", ws, req));
    return;
  }

  try {
    const decision = await wsArcjet.protect(req);

    if (decision.isDenied()) {
      if (decision.reason.isRateLimit()) {
        socket.write("HTTP/1.1 429 Too Many Requests\r\n\r\n");
      } else {
        socket.write("HTTP/1.1 403 Forbidden\r\n\r\n");
      }
      socket.destroy();
      return;
    }

    wss.handleUpgrade(req, socket, head, (ws) => wss.emit("connection", ws, req));
  } catch (e) {
    console.error("WS upgrade error: ", e);
    socket.destroy();
  }
});
```

## Security Benefit
- Protection happens BEFORE WebSocket handshake completes
- Denied connections rejected at HTTP level (cleaner, saves resources)
- No partial WebSocket connection state for denied requests
